# KU D3B Evaluation

This folder contains the in-app evaluation dataset and results for the KU RAG chatbot.

- `datasets/ku_d3b_eval_dataset.jsonl` - default evaluation dataset used by the Evaluation UI tab.
- `results/_last_ui_eval/` - overwritten by each Evaluation UI run.
- `debug/` - reserved for manual debug artifacts.
- `documents/` - Downloaded Documents with the WebCrawler 

Run evaluation from the chatbot UI: open the **Evaluation** tab, choose a dataset/limit, and click **Run Evaluation**.
